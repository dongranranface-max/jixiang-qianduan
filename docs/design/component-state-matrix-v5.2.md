# 组件状态矩阵 v5.2

> 集享公社 UI 设计规范 · 组件状态完整矩阵
> 版本：v5.2 · 2026-05-28

---

## 1. 按钮（Button）

| 状态 | 视觉表现 | 可点击 | 震动反馈 |
|------|---------|--------|---------|
| Default | 渐变背景 + 白字 + 阴影 | ✅ | - |
| Hover | scale(1.02) + shadow-hover | ✅ | - |
| Active/Pressed | scale(0.98) + 阴影减弱 | ✅ | ✅ |
| Loading | 斜条纹动画（btn-flow-load）+ disabled | ❌ | - |
| Disabled | 28% 透明度 + 不可点击 | ❌ | - |
| Success | 绿色背景 + ✓ 图标（可选） | ✅ | ✅ |
| Error | 红色背景 + × 图标（可选） | ✅ | ✅ |

---

## 2. 输入框（Input / FL浮动标签）

| 状态 | 视觉表现 |
|------|---------|
| Default | 浅灰边框($border-color) + placeholder 文字 |
| Focused | 金色边框($gold-on-light) + 外发光 4rpx + label 上浮变小 |
| Filled | label 保持浮动 + 正常边框 |
| Error | 红色边框 + 错误提示文字 + shake 动画 |
| Disabled | 灰色背景 + 不可编辑 + 50% 透明度 |
| Loading | 右侧 loading spinner + 不可编辑 |

## 4. SKU 选择器

| 规格状态 | 视觉表现 |
|---------|---------|
| 未选 | 白底 + 1rpx $border-color 边框 |
| 已选 | $gold 背景 + 白字 |
| 不可选（库存0） | 灰色背景 + 删除线 + 不可点击 |
| Hover | 边框变 $gold-on-light |

---

## 5. 倒计时（Countdown）

| 剩余时间 | 视觉表现 |
|---------|---------|
| ≥ 1 天 | 「N天 HH:mm:ss」正常样式 |
| < 1 天 | 「HH:mm:ss」正常样式 |
| < 1 小时 | 「mm:ss」字体变 $danger |
| < 5 分钟 | 「mm:ss」+ pulse-dot 动画 + 字体 $danger |
| 归零 | 执行 onExpire 回调 |
